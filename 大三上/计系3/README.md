# 计算机系统（3）

- **学期**：大三上（2025-2026 学年第一学期）
- **教材**：《计算机组成与设计：硬件/软件接口》（RISC-V 版）
- **实验平台**：Scala / Chisel、RISC-V 工具链

## 目录结构


## 实验概览

| 实验 | 主题 | 技术点 |
|------|------|--------|
| 1 | MIPS 指令集 | MIPS64 指令格式、汇编编程 |
| 2 | MIPS64 乘法器模拟 | 乘法算法、性能剖析（perf） |
| 3 | 取指和指令译码设计 | Chisel 硬件描述、指令译码逻辑 |
| 4 | 处理器结构实验一 | 五级流水线、流水线冒险 |
| 5 | 处理器结构实验二 | 控制冒险、分支预测 |

## Mini 处理器

`mini/` 目录包含一个完整的 RISC-V 处理器实现（Scala/Chisel），涵盖 Core、Cache、CSR 等核心模块，可供学习和参考。
