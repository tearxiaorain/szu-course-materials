## 实验概览

| 实验 | 主题 | 技术点 |
|------|------|--------|
| 1 | OpenGL 基础图形绘制 | 基本图元、颜色、交互 |
| 2 | 二维变换与观察 | 平移/旋转/缩放、窗口裁剪 |
| 3 | 三维图形与光照 | 3D 变换、光照模型、材质 |
| 4 | 纹理映射与曲线曲面 | 纹理贴图、贝塞尔曲线 |
| 期中大作业 | 俄罗斯方块 | 游戏逻辑、OpenGL 渲染 |
| 期末大作业 | 虚拟场景建模 | 综合场景构建、交互漫游 |

## 运行说明

各实验使用 C++ 和 OpenGL 开发，编译运行方式详见各实验目录下的实验报告。

> ⚠️ 实验代码以 zip 压缩包形式归档，解压后即可编译运行。
