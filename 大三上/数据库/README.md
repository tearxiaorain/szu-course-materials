## 实验概览

| 实验 | 主题 | 技术点 |
|------|------|--------|
| 1 | SQL 的 DDL 语言和单表查询 | CREATE/ALTER/DROP、SELECT、WHERE、聚合函数 |
| 2 | SQL 的多表查询及视图 | JOIN、子查询、VIEW 创建与管理 |
| 3 | 数据库应用 | MySQL/YashanDB 应用开发、SQL 编程 |
| 4 | 数据库设计 | ER 模型、范式化、数据库建模与实现 |

## 说明

实验 3、4 同时支持 MySQL 和国产崖山数据库（YashanDB）两种平台。
