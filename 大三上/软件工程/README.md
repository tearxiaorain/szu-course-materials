## 实验概览

| 实验 | 主题 | 技术点 |
|------|------|--------|
| 1 | 软件界面设计 | UI 设计原则、原型设计 |
| 2 | 数据库建模 | ER 图、PowerDesigner、数据库设计 |
| 3 | 模块过程设计 | 流程图、N-S 图、伪代码 |
| 4 | 模块测试 | 白盒测试、黑盒测试、测试用例设计 |
| 5 | 配置管理 | 版本控制、SVN/Git |
| 6 | 敏捷实践 | TAPD 平台、Scrum、看板管理 |
