## 实验概览

| 实验 | 主题 | 技术点 |
|------|------|--------|
| 1 | 制作标准网线 | 直通线/交叉线、T568A/B 标准 |
| 2 | 常用网络命令 | ping、tracert、ipconfig、netstat、nslookup |
| 3 | Wireshark 抓包分析 | 协议分析、HTTP/DNS/TCP 报文解析 |
| 4 | Socket 编程 | TCP/UDP Socket、客户端/服务器模型 |
| 5 | 交换机与虚拟局域网 | VLAN 配置、交换机管理、生成树协议 |
| 6 | 网络互连与路由 | 静态路由、RIP/OSPF 路由协议 |
| 7 | IPv6 隧道实验 | IPv6 基础、隧道技术、双栈配置 |

## 运行说明

- 实验 1~3、5~7 使用 Cisco Packet Tracer 或 Wireshark 完成
- 实验 4 使用 C/C++ 或 Python 进行 Socket 编程
- 详见各实验目录下的实验指南和报告
