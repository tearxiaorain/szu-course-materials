# 荔荔社区 - 校园社交平台

## 项目概述

荔荔社区是一个基于Web的校园社交平台，采用HTML5、CSS3和原生JavaScript开发。项目采用模块化的架构设计，每个页面都有对应的JavaScript文件来处理特定的功能。

## 页面结构与JS文件分析

### 1. 首页 (index.html)
**链接的JS文件：**
- `src/js/common.js` - 通用功能模块
- `src/js/index.js` - 首页专用功能模块

**功能分析：**

#### common.js (通用模块)
- **返回顶部按钮**：监听滚动事件，当页面滚动超过300px时显示返回顶部按钮
- **下拉菜单**：处理用户信息下拉菜单的显示和隐藏
- **登录状态检查**：检查localStorage中的用户登录状态，更新页面UI
- **UI状态管理**：
  - 已登录用户：隐藏登录/注册按钮，显示用户信息、发布动态区域，启用评论功能
  - 游客状态：显示登录/注册按钮，隐藏用户信息，禁用评论功能
- **关注按钮状态**：根据用户关注列表更新关注按钮的显示状态
- **消息提示**：显示成功、错误、警告等类型的提示消息
- **时间格式化**：将时间戳转换为相对时间显示
- **退出登录**：清除本地存储的用户信息

#### index.js (首页专用模块)
- **内容切换标签**：处理"全站动态"、"关注动态"、"热门推荐"三个标签的切换
- **动态加载**：根据标签类型加载不同类型的动态内容
- **动态交互**：
  - 点赞功能：处理动态的点赞和取消点赞
  - 评论功能：处理动态的评论发布和显示
  - 分享功能：处理动态的分享操作
  - 收藏功能：处理动态的收藏和取消收藏
- **关注用户**：处理关注和取消关注其他用户
- **加载更多**：实现动态的分页加载
- **动态数据管理**：模拟动态数据的获取和渲染

### 2. 登录页面 (login.html)
**链接的JS文件：**
- `src/js/common.js` - 通用功能模块
- `src/js/login.js` - 登录专用功能模块

**功能分析：**

#### login.js (登录专用模块)
- **表单验证**：
  - 学号验证：8-12位数字格式验证
  - 密码验证：最少6位字符验证
- **密码显示/隐藏**：切换密码输入框的显示状态
- **表单提交处理**：
  - 验证所有输入字段
  - 模拟登录请求（测试账号：12345678/password123）
  - 保存用户信息到localStorage
  - 登录成功后跳转到首页
- **消息提示**：显示登录成功、失败等状态消息
- **记住我功能**：处理"记住我"选项的状态

### 3. 注册页面 (register.html)
**链接的JS文件：**
- `src/js/common.js` - 通用功能模块
- `src/js/register.js` - 注册专用功能模块

**功能分析：**

#### register.js (注册专用模块)
- **头像上传预览**：处理头像文件的选择和预览显示
- **表单验证**：
  - 学号验证：8-12位数字格式
  - 姓名验证：最少2个字符
  - 昵称验证：2-15个字符
  - 密码验证：最少6位，必须包含字母和数字
  - 确认密码验证：两次输入必须一致
  - 邮箱验证：标准邮箱格式
  - 验证码验证：6位数字格式
  - 兴趣标签验证：必须选择1-5个标签
  - 用户协议验证：必须同意协议
- **兴趣标签选择**：处理兴趣标签的选择和取消选择
- **验证码发送**：模拟发送验证码，包含60秒倒计时
- **密码显示/隐藏**：切换密码输入框的显示状态
- **表单提交处理**：
  - 验证所有输入字段
  - 模拟注册请求
  - 注册成功后跳转到登录页面
- **消息提示**：显示注册成功、失败等状态消息

### 4. 个人资料页面 (profile.html)
**链接的JS文件：**
- `src/js/common.js` - 通用功能模块
- `src/js/profile.js` - 个人资料专用功能模块

**功能分析：**

#### profile.js (个人资料专用模块)
- **登录状态检查**：检查用户是否已登录，未登录则跳转到登录页面
- **用户资料加载**：从localStorage加载并显示用户资料信息
- **标签页切换**：处理"动态"、"关于我"、"相册"、"好友"四个标签页的切换
- **编辑资料功能**：
  - 模态框显示和隐藏
  - 表单数据填充和验证
  - 头像和封面上传预览
  - 兴趣标签选择
  - 资料更新和保存
- **用户动态加载**：加载并显示用户发布的动态列表
- **用户相册加载**：加载并显示用户的相册内容
- **用户好友加载**：加载并显示用户的好友列表
- **头像和封面上传**：处理头像和封面图片的上传和预览
- **消息提示**：显示操作成功、失败等状态消息

### 5. 发布动态页面 (post.html)
**链接的JS文件：**
- `src/js/common.js` - 通用功能模块
- `src/js/post.js` - 发布动态专用功能模块

**功能分析：**

#### post.js (发布动态专用模块)
- **登录状态检查**：检查用户是否已登录，未登录则跳转到登录页面
- **图片上传功能**：
  - 支持多图片上传（最多9张）
  - 图片格式和大小验证（限制5MB）
  - 图片预览和删除
- **话题标签功能**：
  - 预设话题标签选择（最多3个）
  - 自定义话题添加
  - 话题验证和去重
- **位置选择功能**：
  - 位置模态框显示和隐藏
  - 位置搜索功能
  - 预设位置列表选择
- **隐私设置**：处理动态的隐私级别设置（公开、好友、私密）
- **表单验证**：
  - 内容验证：不能为空
  - 图片验证：格式和大小限制
  - 话题验证：数量限制
- **表单提交处理**：
  - 验证所有输入字段
  - 模拟发布请求
  - 发布成功后跳转到首页
- **取消发布**：确认后返回首页
- **消息提示**：显示发布成功、失败等状态消息

## 技术特点

### 1. 模块化设计
- 每个页面都有独立的JS文件处理特定功能
- `common.js`提供通用的功能模块，避免代码重复
- 清晰的职责分离，便于维护和扩展

### 2. 用户体验优化
- 表单实时验证，提供即时反馈
- 密码显示/隐藏切换
- 图片上传预览
- 消息提示系统
- 加载状态显示

### 3. 数据管理
- 使用localStorage进行本地数据存储
- 模拟后端API接口
- 用户状态管理
- 动态数据缓存

### 4. 交互功能
- 点赞、评论、分享、收藏等社交功能
- 关注/取消关注用户
- 标签页切换
- 模态框交互
- 文件上传

### 5. 响应式设计
- 适配不同设备屏幕
- 移动端友好的交互设计
- 灵活的布局结构

## 使用说明

1. **首页功能**：浏览动态、切换内容类型、进行社交互动
2. **用户认证**：注册新账号或登录已有账号
3. **个人资料**：查看和编辑个人信息、管理动态和相册
4. **发布动态**：创建新的动态内容，支持图片、话题、位置等

## 技术栈

- **前端**：HTML5、CSS3、原生JavaScript
- **图标库**：Bootstrap Icons
- **数据存储**：localStorage
- **开发工具**：现代浏览器开发工具

## 浏览器兼容性

支持所有现代浏览器，包括：
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 项目特色

1. **纯前端实现**：无需后端服务器，可直接在浏览器中运行
2. **模块化架构**：清晰的代码结构，便于理解和维护
3. **完整功能**：包含社交平台的核心功能
4. **用户体验**：注重交互细节和用户反馈
5. **响应式设计**：适配各种设备屏幕
