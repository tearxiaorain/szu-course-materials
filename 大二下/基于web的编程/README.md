## 实验概览

| 实验 | 主题 | 技术点 |
|------|------|--------|
| 1 | HTML 综合实验 | HTML5 标签、表单、多媒体、语义化 |
| 2 | CSS 综合实验 | 选择器、盒模型、布局、响应式设计 |
| 3 | JavaScript 综合实验 | DOM 操作、事件处理、Web Worker、AJAX |

## 大作业

设计并实现一个完整的 Web 应用项目，综合运用 HTML、CSS、JavaScript 技术，包含前端界面设计和交互功能实现。
