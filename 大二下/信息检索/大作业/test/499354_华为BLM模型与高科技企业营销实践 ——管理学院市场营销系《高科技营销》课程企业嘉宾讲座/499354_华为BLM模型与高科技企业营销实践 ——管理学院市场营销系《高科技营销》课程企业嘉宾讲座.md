# 华为BLM模型与高科技企业营销实践 ——管理学院市场营销系《高科技营销》课程企业嘉宾讲座

2023年6月25日下午16:00，应管理学院袁靖波老师邀请，深圳摩彩科技集团总经理周建劼先生来到深圳大学丽湖校区，为深圳大学管理学院市场营销系的《高科技营销》课堂分享了“华为BLM模型与高科技企业营销实践”讲座。周建劼先生拥有华为11年的项目管理与战略规划工作经验，现场听众包括市场营销系2021级01班和02班的同学，课堂气氛热烈，座无虚席。

周总首先结合自身工作经历，首先向同学们介绍了BLM模型的内容，随后通过案例讲解其应用方式。周总以“华为高速增长的背后流程支撑”为切入点，引出流程对企业发展的重要性。周总就DSTE流程中的BLM模型向同学们进行介绍，其中，市场洞察作为BLM模型的核心要素之一，对企业发展至关重要，因此，周总着重对市场洞察进行讲解，包括企业如何通过市场洞察来发现和把握市场机遇。最后，周总向同学们介绍了华为高科技营销实践中的“五看三定”，向同学们讲解了如何通过五个视角的观察找到关键的机遇点，以及如何通过三个层面的分析制定适合企业的产品与服务策略。

在讲座的最后，同学们集体向周建劼经理表达了感谢，袁靖波老师做了简要总结并与周建劼先生合影留念。
