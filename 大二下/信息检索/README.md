## 实验概览

| 实验 | 主题 | 技术点 |
|------|------|--------|
| 1 | 文本处理与索引 | 分词、倒排索引构建 |
| 2 | 位置索引与短语查询 | 位置索引、短语匹配、临近搜索 |
| 3 | 向量空间模型 | TF-IDF、余弦相似度、文档排序 |
| 4 | 索引压缩 | 索引压缩算法、检索效率优化 |
| 5 | Web 检索 | 网络爬虫、PageRank、链接分析 |
| 6 | 文本分类与聚类 | 朴素贝叶斯、K-Means、文本挖掘 |

## 运行说明

各实验使用 Python 开发，详见各实验目录下的实验报告。
