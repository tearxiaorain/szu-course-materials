## 实验概览

| 实验 | 主题 | 核心算法 | 技术点 |
|------|------|----------|--------|
| 1 | 排序算法性能分析 | 快排/归并/堆排序 | 时间复杂度分析、性能对比 |
| 2 | 最近点对问题 | 分治法 | 递归分治、几何算法 |
| 3 | 地图填色问题 | 回溯法 | 图着色、约束满足、剪枝优化 |
| 4 | 鸡蛋掉落问题 | 动态规划 | 状态转移、最优子结构 |
| 5 | 图论——求桥 | Tarjan 算法 | DFS、图的连通性、桥 |
| 6 | 棒球比赛问题 | 最大流 | Ford-Fulkerson、网络流建模 |

## 运行说明

各实验使用 C++ 开发，编译运行方式详见各实验目录下的实验报告。
