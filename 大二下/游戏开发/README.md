## 实验概览

| 实验 | 主题 | 技术点 |
|------|------|--------|
| 1 | 拾苹果 | Unity 基础、场景搭建、碰撞检测、UI |
| 2 | Boom | 物理引擎、粒子系统、音效 |
| 3 | 太空射击 | 预设体、对象池、游戏状态管理 |
| 大作业 | 新春跑酷 | 完整游戏开发流程、团队协作、迭代开发 |

## 运行说明

使用 Unity Hub 打开各实验/大作业项目目录，选择对应 Unity 版本打开即可。演示视频展示了运行效果。
