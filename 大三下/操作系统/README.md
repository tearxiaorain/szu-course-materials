- **实验平台**：Linux（xv6-riscv）、Ubuntu 20.04

## 目录结构

```

    ├── 3/          # 实验三 + 综合实验一：xv6入门 + 调度 + 信号量
    ├── 4/          # 实验四：内存分配与回收
```

## xv6 说明

xv6-riscv 是 MIT 开发的教学操作系统，用于学习操作系统内核原理。本目录包含实验过程中修改的版本。

原始项目地址：`git://g.csail.mit.edu/xv6-labs-2023`

## 运行说明

xv6 在 Ubuntu 20.04 环境下编译运行：
```bash
# 编译
make qemu

# 调试
make qemu-gdb
```

其他实验在 Linux 环境下用 gcc 编译运行。
