# 系统编程

- **学期**：大三下（2025-2026 学年第二学期）

## 目录结构

```
系统编程/
├── README.md
└── 实验/
    ├── 1/          # 实验一：进程控制（fork/守护进程/僵尸进程）
    ├── 2/          # 实验二：进程间通信
    │   ├── calapplication/  # 计算器应用（socket通信）
    │   └── chatsystem/      # 聊天系统（socket通信）
    ├── 3/          # 实验三：多线程编程
    ├── 4/          # 实验四：文件控制
    └── 大作业/      # 综合大作业：聊天系统
        └── code/   # 项目源码
            ├── conf/
            ├── include/
            └── src/
```

## 实验概览

| 实验 | 主题 | 技术点 |
|------|------|--------|
| 1 | 进程控制 | fork、守护进程、僵尸进程、exec |
| 2 | 进程间通信 | 信号、管道、socket、GDB多进程调试 |
| 3 | 多线程编程 | 线程创建、同步、互斥 |
| 4 | 文件控制 | IO函数、文件描述符、内核文件表 |
| 大作业 | 综合项目 | 聊天系统（socket通信） |

## 运行说明

各实验在 Linux 环境下用 gcc 编译运行，详见各实验目录下的实验报告。
