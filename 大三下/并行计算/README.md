## 实验概览

| 实验 | 主题 | 并行框架 | 关键技术 |
|------|------|----------|----------|
| 1 | 矩阵乘法 | MPI | 6个基本MPI语句、进程间通信、加速比分析 |
| 2 | Cannon矩阵乘法 | MPI | MPI_Isend/Irecv/Wait、非阻塞通信、棋盘划分 |
| 3 | 高斯-约旦消去法 | MPI | MPI_Allgather/Bcast/Reduce、选主元 |
| 4 | 前缀和 | Pthreads | 条件变量/信号量、共享内存、临界区控制 |
| 5 | 寻找完数 | Pthreads | 数据竞争、负载平衡、同步语句 |
| 6 | PSRS排序 | OpenMP | 数据划分、并行排序、PSRS算法 |
| 7 | 矩阵乘法 | CUDA | GPU编程、归约求和、线程束洗牌 |
| 8 | 双调排序 | CUDA | GPU架构、并行双调排序算法 |

## 运行说明

所有实验在深圳大学 HPC 平台上运行，远程登录方式：
```bash
# MPI/Pthreads/OpenMP 实验
ssh bxjs@hpc.szu.edu.cn

# CUDA 实验（先登录 HPC，再跳转）
ssh bxjs@hpc.szu.edu.cn
ssh CUDA
```

各实验独立编译运行，编译命令详见实验报告。
